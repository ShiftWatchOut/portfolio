# Ubuntu 再备忘

曾经为了装酷而接触 Ubuntu ，但当时还不明白酷在哪里，觉得“小众=极客”。两年的工作中，所就职的公司给配了 MacBook，在终端里自由探索，在工作区间自由切换，开发体验确实友好。但考虑到公司配的电脑总归要还回去，还是给自己电脑一个类似的开发体验吧。鉴于苹果操作系统与 Linux 的相似性，终于还是回到了 Ubuntu。期望通过记录下这些优化体验的方法，让我在不使用 Mac 时，也能拥有类似的体验。啊？你问为啥不直接买 MacBook ？拜托，谁不知道这玩意有品牌加持的溢价，我这种实用主义者会为了这种自己动动手就能轻松获得的体验付费吗。自己动手，丰衣足食。

### 1. 用户文件夹名中英调整

Ubuntu 有个很奇特的 feature ，操作系统语言被切换之后，用户文件夹下的“桌面”“视频”“图片”之类的文件夹也会被重命名成对应的语言，这就导致，如果我们安装时使用的语言是中文的，如果想在终端里很快地跳转到用户文件夹下各个子目录，便需要中英切换。有两种方法可以调整：

* 执行下面的命令，然后编辑

```sh
gedit ~/.config/user-dirs.dirs
```

* 图形界面操作

设置 -> 区域与语言 -> 选择英语并注销 -> 重新登录 -> 在弹窗询问时选择“更新名称” -> 语言切换回中文 -> 这次选择“保留旧名称”

### 2. Clash for Windows

> 下载地址：[clash_for_windows_pkg](https://github.com/Fndroid/clash_for_windows_pkg/releases)

压缩包解压到某个文件夹，双击 cfw，然后设置开机启动，以后怎么启动就不用管了。在 Dashboard 里设置转发端口，一般是 7890，然后再：

设置 -> 网络 -> 代理改为手动 -> 一致设为 `http://127.0.0.1:7890`
